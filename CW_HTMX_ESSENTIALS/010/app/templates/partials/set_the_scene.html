Welcome to the Modern Airport Terminal!

In this lesson, our web application is an airport terminal. The central feature, the Live Departures Board, is our primary data display. Like a real airport board, it needs to show the latest information without requiring you to constantly ask an agent (i.e., reload the page).

We'll see how the board updates automatically at regular intervals (polling), and how an urgent "Gate Change" announcement can trigger an immediate, out-of-band refresh. We'll also explore how clicking a flight to view its details provides a seamless update, much like using a terminal kiosk, without navigating away from the main hall. Finally, we'll see how scanning a boarding pass can result in the server redirecting you to a completely different area.

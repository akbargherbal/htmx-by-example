Welcome to the Lush Community Garden!

Think of this web application as a digital community garden. It's a dynamic, living space where you can cultivate data just like you would plants. Every action you take has a direct and immediate impact on the garden's state, without needing to refresh the entire landscape.

In this lesson, we'll connect gardening actions to core web development concepts. Planting a new seed is like making a POST request to create new data. Replacing an existing plant with another is a PUT request to update it. And pulling a pesky weed is a DELETE request to remove unwanted data.

You'll also see how the garden takes care of itself. The "Garden Status" panel automatically checks for weeds every few seconds, demonstrating how HTMX can poll a server for updates. This is how modern web apps stay fresh and responsive, just like a well-tended garden.
